Ejemplo instalado ok

*La terminal espera el archivo de configuracion:
$ git config --global core.editor "code --wait"

El comando nos abre vs code:
$ git config --global -e




Configuración core.autocrlf:


Se utiliza para eliminar el salto de linea entre sistemas operativos:

$ git config --global core.autocrlf input/true


el comando entrega listado de funciones:

git config -h:



  INICIO:Comandos git:


Navegacion en la consola

code . : Abre el editor de texto.    
ls : lista contenido del directorio.
ls -a : lista contenido del directorio incluyendo archivos ocultos.
pdw : lista directorio actual.
cd : Moverse a un directorio.
cd .. : Devolverse a un directorio.
mkdir: Crear una carpeta.




Comandos git

git init :Inicia carpeta por defecto.
git config --global init.defaultBranch main:Cambia el nombre de la rama principal.
git status :  Estado actual de los archivos en fichero git.
git add opcion1 opcion2 ... : Agrega un archivo .
git add . : Agrega todos los archivos del fichero .


Mensajes de la consola


Untracked files:Archivos no seguidos.
Changes to be committed:Cambios listos para comprometer .




    
    
  INICIO:Flujo de trabajo Git



    
STAGE

    git add opcion1 opcion2 ... : Agrega un archívo .
    git add . : Agrega todos los archívos del fichero .

    
COMMIT

    git commit:Compromete los cambios.
    git commit -m "comentario" :Compromete los cambios, más un comentario.
    git rm --cached ... :Retira cambios del commit.
    git log :Muestra los commit realizados.
    git log --oneline :Muestra los commit realizados en una sola linea.
    git checkout :Apunta a un archio o fichero.

    
SERVER


    …or create a new repository on the command line

    echo "# prueba" >> README.md
    git init
    git add README.md
    git commit -m "first commit"
    git branch -M main
    git remote add origin https://github.com/warnerxd/prueba.git
    git push -u origin main



    …or push an existing repository from the command line

    git remote add origin https://github.com/warnerxd/prueba.git
