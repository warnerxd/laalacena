Ejemplo instalado ok

*La terminal espera el archivo de configuracion:
$ git config --global core.editor "code --wait"

El comando nos abre vs code:
$ git config --global -e




Configuración core.autocrlf:


Se utiliza para eliminar el salto de linea entre sistemas operativos:

$ git config --global core.autocrlf input/true


el comando entrega listado de funciones:

git config -h:
